# nscc-mscloud-resources

Resources, guides, scripts, and lab examples for the **NSCC Visual Studio Enterprise Subscription Pilot**.

This repo is maintained by NSCC IT and is intended to help students and staff get the most out of the Azure credits and developer tools included in the subscription.
