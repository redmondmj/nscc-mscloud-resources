# Labs

Hands-on exercises for students in the NSCC Visual Studio Enterprise Subscription Pilot.

Each lab folder contains a step-by-step guide and any supporting scripts or templates.

## Lab list

| Lab | Topic | Difficulty |
|-----|-------|------------|
| [Lab 01 — VM Deployment](lab-01-vm-deployment/) | Deploy a Linux VM using the Azure Portal and connect via SSH | Beginner |
| [Lab 05 — Governance, Cost and Guardrails](lab-05-governance-cost-guardrails/) | Azure Policy, resource locks, cost analysis, budgets, and the Pricing Calculator | Intermediate |
| [Lab 06 — M365 Administration](lab-06-m365-saas/) | Entra ID, Exchange Online, Defender XDR, SharePoint, and Microsoft Teams admin tasks | Intermediate |
| [Lab 07 — Intune Endpoint Management](lab-07-intune/) | Entra ID join, configuration profiles, app deployment, antivirus policy, and PowerShell via Intune | Intermediate |

More labs coming soon. See [CONTRIBUTING.md](../CONTRIBUTING.md) to submit a lab.
